Página MedSchedule

Requisitos:
- Python 3.9

Instalación:
1. Descomprimir archivo
2. Abrir CMD e introducir "py -m venv venv"
3. En la misma ventana introducir ".\venv\Scripts\activate"
4. Luego introducir "py -m pip install --upgrade pip"
5. Introducir "py -m pip install -r requirements.txt"
6. Finalmente iniciar el servidor "py .\manage.py runserver"

El sitio no tiene ningún administrador o login.
